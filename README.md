# 🌟 LuminorGui

**LuminorGui** — это мощная и стильная open-source библиотека для создания интерфейсов в скрипт-хабах Roblox. Она позволяет быстро собирать красивые меню с вкладками, переключателями и кнопками, экономя ваше время и нервы.

## 🚀 Возможности

- **Красивая заставка** с Pop-up эффектом при запуске.
- **Главное окно** с возможностью перетаскивания, изменения размера и сворачивания[reference:0].
- **Вкладки (Tabs)** с неоновой подсветкой активной вкладки[reference:1].
- **Переключатели (Toggles)** с запоминанием состояния и цветной неоновой полоской[reference:2].
- **Кнопки (Buttons)** с анимацией расходящейся волны (ripple-эффект)[reference:3].
- Простая установка через `loadstring`.
